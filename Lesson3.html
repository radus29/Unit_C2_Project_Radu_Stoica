Comments

Comments are text hidden within an HTML document. In order to create an HTML comment, you must use the characters "<!--" before the comment text and "-->" after it. Example: <!-- This is a single line comment -->

Attributes

Attributes are codes that appear within a tag that modify it in some way.
The general syntax for an attribute is:
<tagName attributeName="value">content</tagName>
Example: <img src="happyface.jpg"> <p style="color:red">This paragraph will have red text</p>

Links use the anchor or <a> element to make a hyperlink to another page or to another location on the same page.
The general syntax for a link is <a href="pathToFile">LinkText</a>
Example: <a href="https://google.com"> Google</a>

Images

Images are embedded on a web page using the <img> element. It is an empty element (no trailing tag) and works inline. To locate the image, you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.
Local image: <img src="happyface.jpg" alt="Smiley Face">
Remote image: <img src="https://ex.com/hf.jpg" alt="Smiley Face"> To size an image, use <img src="file" width="200">
