Formatting Tags - elements that modify the look of text within them

Examples of formatting tags

  • <b>This will in bolded text</b>
  • <strong>This will appear in bold</strong>
  • <i>This will appear in italics</i>
  • <mark>This will appear highlighted</mark>
  • <small>This text will be smaller</small>
  • <del>This will appear crossed out</del>
  • <ins>This will appear underlined</ins>
  • <sub>This will appear as subscript</sub>
  • <sup>This will appear as superscript</sup>

Tags to create lists

  • <ol> creates an ordered list (a list with numbers).
  • <ul> creates an unordered list (a list with bullets instead of numbers).
Both list types use <li> (list item) for the items within the lists.

Example:

<ol>
  <li>list</li>
  <li>is</li>
  <li>numbered</li>
</ol>

<ul>
  <li>list</li>
  <li>has</li>
  <li>bullets</li>
</ul>
