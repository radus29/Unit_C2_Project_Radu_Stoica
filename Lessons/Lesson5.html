Data tables

Tables allow web developers to arrange data into a grid-like structure.
The essential elements for tables are:

  1. table: creates the table
  2. tr: creates a row
  3. th: creates a heading cell
  4. td: creates a data cell

You can force a table (or the cells within it) to have specific sizes. You can specify pixels (px) or percentage of the width/height of the page. Use the following code to do so: 

table {
  width:100%;
  height:500px;
}
th,td {
  width:100px;
  height:50px;
}
to break a cell across more than one row or column (or both), use the following attributes: <th colspan="2"> <td rowspan="2">

Example of a table in HTML: 

<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
