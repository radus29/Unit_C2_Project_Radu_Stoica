Home Page Lesson 2

HTML stands for HyperText Markup Language, the language that makes the World Wide Web possible. HTML was invented by Tim Berners-Lee in 1989.
Every html code begins with <!DOCTYPE html>, <html> and ends with </html>

Example HTML code: 

<!DOCTYPE html>
<html>
<head>
  <title>Title of Web Page</title>
</head>
<body>
  <h1>Main heading of Web Page</h1>
  <p>Paragraph content uses the p
  tag.</p>
</body>
</html>
