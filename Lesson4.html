Headings

Headings are used to create an outline structure on a web page.
There are 6 heading types: <h1> through <h6>.
Example: 

  1. <h1>The top-level heading</h1>
    2. 
  
  2. <h2>Second level</h2>
    3. 
    
  3. <h3>Third level</h3>
    4. 
      
  4. <h4>Fourth level</h4>
    5. 
        
  5. <h5>Fifth level</h5>
    6. 
          
  6. <h6≶Sixth level</h6>
    7.

Paragraphs

Paragraph is the workhorse tag of web design. Most blocks of text use paragraphs for their formatting.
Example:
<p> Most text in web pages are formatted in paragraphs. </p>
If you want to create a line between blocks of text, use the <hr> element. hr is an empty tag.
